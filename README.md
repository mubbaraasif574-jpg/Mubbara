# Birthday Wishes 🎂🎉

A beautiful React web application where you can send and receive birthday wishes!

## Features

✨ **Features Included:**
- Create birthday wishes for someone special
- Add your name as the sender
- View all birthday wishes that have been sent
- Delete wishes if needed
- Beautiful, responsive UI with smooth animations
- Works on desktop and mobile devices

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (Node Package Manager)

### Installation

1. Navigate to the project directory:
```bash
cd mubbara
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

Start the development server:
```bash
npm start
```

The application will open in your browser at `http://localhost:3000`

### Build for Production

Create an optimized production build:
```bash
npm run build
```

## How to Use

1. **Fill the Form:**
   - Enter the birthday person's name
   - Enter your name
   - Write your birthday wish message

2. **Send the Wish:**
   - Click "Send Wish 💌" button
   - Your wish will be added to the collection

3. **View Wishes:**
   - Click "View Wishes" button to see all wishes sent
   - Each wish shows the recipient's name, your message, and the date

4. **Add More Wishes:**
   - Click "Add New Wish ➕" to create another wish

5. **Delete a Wish:**
   - Click the "✕" button on any wish card to remove it
